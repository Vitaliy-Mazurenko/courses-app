# courses-app

React Task 

```
1. Clone repo with github:
  git clone https://github.com/Vitaliy-Mazurenko/courses-app.git

2. Go to server project folder(backend), install dependencies and run server: npm start
   Now you have a backend for application. Go to the url: http://localhost:4000/api

3. Go to client project folder(frontend), install dependencies and run client: npm start
   Now you have a frontend for application. Go to the url: http://localhost:3000/

4. Admin credentials. These credentials provide role ADMIN
```
